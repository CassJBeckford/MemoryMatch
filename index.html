MEMORY MATCH


Memory Match is a game to test your memory

Once the game has started the gameboard will feature a set amount of cards depending on the difficulty.

All cards contain a picture placed face down that will be revealed once pressed.

Each picture appears on a card exactly twice and in order to make a match you must turn over the cards with the same picture.

If you turn over a pair of cards that don't match they will both return to their face down position.
